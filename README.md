```bash
  npm install viewers6

```
Application
dir: viewer5/
npm start

Server
dir: viewer5/viewer5-service
npm start app.js
    
## API Reference

#### Get all items

```http
  GET /api/data
```

#### Get item

```http
  GET /api/data/id
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
